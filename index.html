Welcome to MedianShape’s documentation!¶

MedianShape is a simple Python package for solving Simplical Median Shape Problem (SMSP) in simplicial settings using simplicial flat norm distance. It contains several experiments in 2D and 3D. MedianShape is distributed under the GNU GPL.
